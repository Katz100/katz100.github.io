Mobile SDKs

What is an SDK?

An SDK (Software Development Kit) is a collection of tools, libraries, documentation, and APIs that help developers build applications for a platform. Mobile SDKs provide access to device features like the camera, GPS, sensors, notifications, and storage.

Examples
  • iOS SDK: Includes SwiftUI/UIKit, networking, Core Data, and device APIs.
  • Android SDK: Includes Jetpack libraries, Compose, Activities/Intents, and device APIs.
